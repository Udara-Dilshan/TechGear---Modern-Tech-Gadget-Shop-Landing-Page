## ğŸš€ Getting Started

### Prerequisites

- Node.js (v16 or higher)
- npm or yarn

### Installation

1. Clone the repository
```bash
git clone [REPOSITORY_URL]
cd techgear-landing-page
```
2.
Install dependencies
```
npm install
````

3.Start the development server
```
npm run dev
````

4.Open http://localhost:5173 in your browser
